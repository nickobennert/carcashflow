Zusammenfassung

Kernaussage

Die Carcashflow-Plattform verwendet ein intelligentes Routensystem zur Berechnung von Fahrtstrecken und zur Ermittlung passender Mitfahrgelegenheiten. Die monatlichen Kosten für diese Infrastruktur liegen zwischen €0 und €30, abhängig vom gewählten Modell.

Aktueller Stand

Entwicklung Demo-Server

Die Anwendung nutzt derzeit den öffentlichen OSRM-Demo-Server. Dieser ist für Entwicklung und Tests geeignet, jedoch nicht für den Produktivbetrieb zugelassen.

Was ist OSRM?

OSRM (Open Source Routing Machine) ist eine quelloffene Software zur Berechnung von Fahrtrouten. Sie verwendet OpenStreetMap-Kartendaten und berechnet:

  • Kürzeste/schnellste Route zwischen zwei Punkten
  • Fahrzeit und Distanz
  • Wegpunkte entlang der Strecke

Wichtig: OSRM selbst ist kostenlos. Die Kosten entstehen durch den Server, auf dem OSRM betrieben wird.

Hosting-Optionen

Empfohlen Option 1: Self-Hosting

Ein eigener Server wird angemietet und OSRM darauf installiert.

Aspekt Details
Anbieter Hetzner, DigitalOcean, AWS
Empfohlene Specs 4 GB RAM, 2 CPU Cores, 50 GB SSD
Monatliche Kosten €10 - €30
Kartendaten Deutschland/DACH (~2-4 GB)

Vorteile:

  • Vollständige Kontrolle
  • Keine Anfragelimits
  • DSGVO-konform (Server in Deutschland möglich)
  • Vorhersehbare, fixe Kosten

Option 2: Managed Services

Externe Anbieter betreiben die Routing-Infrastruktur.

Anbieter Kosten Bemerkung
OpenRouteService Kostenlos* Bis 2.000 Anfragen/Tag, DSGVO-konform
MapBox Directions ~€0,50 / 1.000 Anfragen US-Anbieter, sehr schnell
Google Routes API ~€5 / 1.000 Anfragen Teuer, aber zuverlässig

Häufige Fragen

Wer stellt die Rechnung?

  • Self-Hosting: Server-Anbieter (z.B. Hetzner) → Monatliche Rechnung
  • Managed Service: API-Anbieter → Nutzungsbasierte Rechnung

Kann ich später wechseln?

Ja, die Anwendung ist so gebaut, dass der Routing-Anbieter ausgetauscht werden kann. Kein Datenverlust, da Routen in eigener Datenbank gespeichert werden.

Datenschutz (DSGVO)

  • Self-Hosting: Alle Daten bleiben auf eigenem Server - DSGVO-konform
  • OpenRouteService: Deutsches Unternehmen - DSGVO-konform
  • US-Anbieter: Datenschutzhinweis in AGB erforderlich

Nächste Schritte

  1. Entscheidung für Hosting-Modell treffen
  2. Budget festlegen (€0-30/Monat realistisch)
  3. Timeline für Produktivgang bestimmen
  4. Umsetzung durch Entwicklungsteam
